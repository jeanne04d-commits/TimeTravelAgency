# TimeTravel Agency - Webapp Interactive

## 🎯 Description

TimeTravel Agency est une webapp immersive permettant aux utilisateurs de découvrir et de réserver des voyages dans différentes époques historiques.

L’utilisateur peut explorer plusieurs destinations, interagir avec un assistant IA et obtenir des recommandations personnalisées.

---

## 🛠️ Technologies utilisées

- Next.js (React)
- Tailwind CSS
- v0.dev (génération de l’interface)
- Chatbase (chatbot IA)
- Vercel (déploiement)

---

## ✨ Fonctionnalités

- Landing page immersive (design dark mode premium)
- Hero section avec animation
- Galerie de 3 destinations :
  - Paris 1889
  - Crétacé (-65 millions d’années)
  - Florence 1504
- Cards interactives avec visuels intégrés
- Chatbot IA conversationnel (Conseiller Temporel)
- Recommandations personnalisées
- FAQ via chatbot
- Formulaire de réservation (simulation)
- Site responsive (mobile-first)

---

## 🤖 Outils IA utilisés
